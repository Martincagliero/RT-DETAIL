# Imagen - Sección Motos

Coloca aquí la imagen principal para la sección de motocicletas.

## Imagen requerida:
- `motorcycle.jpg` - Moto trabajada/detallada

## Especificaciones:
- **Formato**: JPG o WebP
- **Aspecto**: 3:4 (vertical)
- **Resolución**: Al menos 900x1200px
- **Peso**: Optimizado (< 500KB)
- **Contenido**: Motocicleta limpia/detallada en ambiente elegante

## Tips:
- Fondo oscuro o neutro
- Buena iluminación que resalte los brillos
- Ángulo que muestre el trabajo de detailing
- Puede ser una moto deportiva, custom, o clásica
