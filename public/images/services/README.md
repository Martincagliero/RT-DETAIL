# Assets - Imágenes de Servicios

Coloca aquí las imágenes para cada servicio.

## Imágenes requeridas:
- `opticos.jpg` - Trabajo en faros/ópticos
- `sellados.jpg` - Sellado cerámico de pintura
- `tapizados.jpg` - Limpieza/restauración de tapizados
- `ruedas.jpg` - Detailing de llantas
- `interiores.jpg` - Limpieza de interiores
- `chasis.jpg` - Trabajo en chasis/bajos
- `motor.jpg` - Detailing de motor

## Especificaciones:
- **Formato**: JPG o WebP
- **Aspecto**: 4:3 (horizontal)
- **Resolución**: Al menos 1200x900px
- **Peso**: Optimizado (< 500KB por imagen)
- **Calidad**: Alta calidad, muestra el antes/durante/después

## Tips:
- Usa imágenes que muestren claramente el trabajo realizado
- Buena iluminación
- Enfoque en los detalles
- Si no tienes imágenes, se mostrará un gradiente placeholder
